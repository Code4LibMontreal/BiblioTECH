# Applying to the BiblioTECH jumpstart program
return to [home page](https://claraturp.github.io/BiblioTECH/)

## About the process
We welcome all applications from any student currently enrolled in a Masters of Information Studies program in Quebec, Canada. To show interest in this program, you must fill the application form, which is available below under important links. Please note that there will be no interviews following the application process to select participants, please make sure you show us why you are interested in this program and why we should select you. Finally, we are committed to creating a diverse and accessible environment for this program.

### What you need to apply?
* Current CV
* Fill out the application form

## Important dates:

* May: deadline for applications
